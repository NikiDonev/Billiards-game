# Billiards game


<p>A simple billiards game made with C++ and SFML.</p>
<br>

## Controls
| | |
|-|-|
|Left click and hold | Aim and shoot|
|Right click and hold | Move the ball |
| R | Reset game |
| S | Singleplayer mode|
| M | 2 player mode |
| Numbers 1 to 5 | Switch theme |
